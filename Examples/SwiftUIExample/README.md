# Auralog SwiftUI Example

This example shows the recommended one-line startup integration, SwiftLog routing, direct structured logs, and `Auralog.task` error capture.

Run it with:

```bash
AURALOG_API_KEY=aura_your_key swift run --package-path Examples/SwiftUIExample
```

For iOS apps, use the same initialization pattern inside your `@main App.init()`.
